Privacy Policy - Privacy Inspector

Privacy Inspector does not collect, store, or transmit any user data.

Data Collection

This extension does not collect any personal information, browsing history, or usage data.

How It Works

  • All processing happens locally in your browser
  • No data is sent to external servers
  • No tracking or analytics
  • Only accesses data when you actively click the extension icon
  • Only sees the specific website you're currently viewing

Permissions Explanation

  • activeTab - Accesses the current tab only when you click the extension
  • cookies - Views and manages cookies for the current website only
  • scripting - Reads storage data from the current website

Open Source

This extension is open source. View the code on GitHub.
